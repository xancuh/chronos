# Bot System:

> [!NOTE]
> Please do not publicly use the bot's source as it is insecure and full of bugs. I may edit this source to match the current one that's being used.

First, you want to go to fps.ms and register using Discord. This is what I use for my bots, and its really good. They also have more services, but that's not about this.

Go and go to the "Deployemnt Container", choose "Discord Bot" and choose the Python package then scroll down and press: "Continue to Configuration".
Here, you will be able to set the server location and the server name. Choose your location.

After deploying, you will want to go to your dashboard, click on the server and click on files on the sidebar. Press "Upload" and upload the files. Each one.

Now, here's where it gets more tasky.

- Create a bot through the Discord Developer Portal and get the token. Paste the token in the config where it says something with the bot_token.
- Set your prefix. Could be ```!```, ```:``` etc. Make sure it is a symbol or something.
- Change the logs thing to a channel ID of your choice. This is where the keys are shown to staff.

Go to the console, press "Start" and it should work if you've done everything successfully!

I will not be replying to any issues, fps.ms has a AI assistant that should help you.
